# Barangay Processes Knowledge Base

## Overview
Barangays are the smallest administrative units in the Philippines. They provide frontline government services, maintain peace and order, and handle community-level disputes and documentation.

---

## 1. Barangay Clearance

### Description
A Barangay Clearance is a document issued to certify that a resident has no pending legal or administrative issues within the barangay.

### Requirements
- Valid ID
- Proof of residency
- Application form

### Steps
1. Proceed to the barangay hall
2. Fill out the application form
3. Submit requirements
4. Pay the required fee
5. Wait for processing
6. Receive clearance

### Processing Time
- Typically 10–30 minutes

### Fees
- Usually PHP 20–100

---

## 2. Certificate of Indigency

### Description
Issued to residents who are financially disadvantaged and need assistance for medical, legal, or educational purposes.

### Requirements
- Valid ID
- Proof of residency
- Request letter (optional)

### Steps
1. Request certificate at barangay hall
2. Provide necessary details
3. Verification by barangay officials
4. Issuance of certificate

### Processing Time
- Same day

---

## 3. Barangay Business Clearance

### Description
Required before starting or renewing a business. Confirms that the business complies with barangay regulations.

### Requirements
- DTI/SEC registration
- Valid ID
- Lease contract or proof of business location
- Previous clearance (for renewal)

### Steps
1. Submit application form
2. Provide required documents
3. Inspection (if required)
4. Pay fees
5. Receive clearance

### Processing Time
- 1–3 days

---

## 4. Blotter Report

### Description
An official record of incidents or complaints filed at the barangay.

### Requirements
- Valid ID
- Personal statement of incident

### Steps
1. Report incident to barangay officer
2. Provide detailed account
3. Incident is recorded in blotter logbook
4. Receive copy if requested

### Processing Time
- Immediate recording

---

## 5. Barangay Dispute Resolution (Lupon Tagapamayapa)

### Description
Handles mediation and conciliation of disputes between residents.

### Steps
1. File complaint at barangay
2. Schedule mediation meeting
3. Conduct hearing with both parties
4. Reach agreement or settlement
5. Issue certification if unresolved

### Possible Outcomes
- Amicable settlement
- Referral to court (with Certificate to File Action)

### Processing Time
- Several days to weeks

---

## 6. Certificate of Residency

### Description
Certifies that a person is a resident of the barangay.

### Requirements
- Valid ID
- Proof of residency

### Steps
1. Request certificate
2. Verification by barangay staff
3. Issuance of certificate

### Processing Time
- Same day
